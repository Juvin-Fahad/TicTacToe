# 🎮 Terminal-Based Tic Tac Toe Game (Java)

This is a terminal-based Tic Tac Toe game written in Java. It features a simple user registration system before the game starts.

---

## ✅ Features

- 🔐 **User Registration**:  
  Users must register before starting the game.

- 👤 **View Registered Info**:  
  Users can view their registered details before playing.

- 🕹️ **Tic Tac Toe Gameplay**:  
  Classic 2-player turn-based game in the terminal.

- 🔁 **Game Start Option**:  
  After registration, users can choose whether to start the game (`y/n`).

- ❌ **Exit Option**:  
  Exit the program from the main menu at any time.

---

## 📋 Menu Options
    Register
    Exit
    
  🏁 Gameplay

    The game board is a 3x3 grid.

    Player 1 uses O, Player 2 uses X.

    Players input row and column numbers (1 to 3) to mark their moves.

    The game detects wins or a draw automatically

  🛠️ Requirements

    Java JDK 8 or higher

    Terminal / Command Prompt
